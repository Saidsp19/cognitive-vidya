# Python Course Overview

Welcome to the **complete Python programming course** — from your first line of code to advanced, production-ready patterns.

<VideoEmbed videoId="kqtD5dpn9C8" title="Python for Beginners — Full Course Introduction (freeCodeCamp)" />

## Who Is This For?

| Level | You Will Learn |
|-------|----------------|
| **Beginner** | Variables, types, control flow, and core data structures |
| **Intermediate** | Functions, OOP, files, error handling, comprehensions |
| **Advanced** | Decorators, generators, testing, async, and best practices |
| **Expert** | DSA patterns for interviews and competitive programming |

No prior programming experience is required. Each lesson builds on the previous one.

## How to Use This Course

Every module follows the same rhythm:

1. **Read the theory** — Concepts explained with examples and diagrams
2. **Practice in Colab** — Click the badge on each lesson to open an interactive notebook
3. **Pass the auto-grader** — Complete exercises until all tests pass

:::tip Recommended Path
Follow the sidebar order from top to bottom. Complete each notebook before moving on.
:::

## Course Modules

### Module 1: Fundamentals
Python setup, variables, types, operators, and input/output.

### Module 2: Control Flow
`if`/`else`, `for` loops, `while` loops, and logical thinking.

### Module 3: Data Structures
Lists, tuples, sets, dictionaries, and string manipulation.

### Module 4: Functions
Defining functions, scope, closures, and functional programming.

### Module 5: Object-Oriented Programming
Classes, inheritance, polymorphism, and special methods.

### Module 6: Files & Error Handling
Exceptions, reading/writing files, JSON, and CSV.

### Module 7: Intermediate Python
Comprehensions, generators, decorators, and context managers.

### Module 8: Advanced Python
Modules, testing, type hints, async basics, and best practices.

### Module 9: DSA Patterns
Pattern-based problem solving for interviews — Sliding Window, HashMap, Stack, Binary Search, and more.

## What You'll Build

By the end of this course you will be able to:

- Write clean, idiomatic Python for real-world tasks
- Design programs with functions and object-oriented patterns
- Handle files, APIs, and errors confidently
- Solve algorithmic problems using reusable patterns
- Read and contribute to professional Python codebases
